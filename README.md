 🎵 Aura Music Bot

**Продвинутый музыкальный Telegram бот** с поиском треков, функцией Shazam и персональными плейлистами. Создан для удобного прослушивания и обмена музыкой в мессенджере.

## ✨ Возможности

### 🎵 **Музыкальные функции**
- **🔍 Умный поиск** - находите треки по названию или исполнителю
- **🎧 Shazam** - распознавание музыки по голосовому/аудио
- **⚡ Кэширование** - популярные треки загружаются мгновенно
- **🎬 Высокое качество** - загрузка в лучшем доступном формате (192 kbps)

### 👤 **Персонализация**
- **❤️ Избранное** - создавайте личные плейлисты
- **📂 Коллекция** - быстрый доступ к сохраненным трекам
- **🚀 Шеринг** - делитесь музыкой с друзьями

### 🛠️ **Администрирование**
- **📊 Статистика** - подробная информация о пользователях и треках
- **📢 Рассылки** - уведомления для всех пользователей
- **� Безопасность** - защита данных и конфиденциальности

## 🚀 Быстрая установка

### 1. Клонирование репозитория
```bash
git clone https://github.com/yourusername/aura-music-bot.git
cd aura-music-bot
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка конфигурации
```bash
# Скопируйте шаблон конфигурации
cp .env.example .env

# Отредактируйте .env файл
nano .env  # или используйте любой текстовый редактор
```

### 4. Заполните .env файл
```env
# Telegram Bot Configuration
BOT_TOKEN=1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
ADMIN_ID=123456789
BOT_USERNAME=your_bot_name

# Directory Configuration  
CACHE_DIR=temp_music
LOG_FILE=bot_log.txt

# Database Configuration
USERS_DB=users.db
MUSIC_DB=music_data.db
```

### 5. Запуск бота
```bash
python main.py
```

## 🔧 Настройка

### 🤖 Получение токена бота
1. Найдите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте полученный токен

### 👤 Получение Admin ID
1. Найдите [@userinfobot](https://t.me/userinfobot) в Telegram
2. Отправьте ему любое сообщение
3. Скопируйте ваш User ID

### 🎬 Установка FFmpeg (обязательно для аудио)

**Windows:**
```bash
# Скачайте с https://ffmpeg.org/download.html
# Распакуйте и добавьте в PATH переменные окружения
```

**Linux (Ubuntu/Debian):**
```bash
sudo apt update && sudo apt install ffmpeg
```

**macOS:**
```bash
brew install ffmpeg
```

## 🎮 Команды бота

### 📱 Для всех пользователей:
- `/start` - Запуск бота и приветствие
- `/playlist` - Показать личный плейлист

### 👨‍💻 Для администратора:
- `/stats` - Статистика бота (пользователи и треки в кэше)
- `/broadcast` - Создание рассылки пользователям

## 💡 Использование

### 🔍 Поиск музыки
Просто напишите название трека или исполнителя в чат:
```
Queen Bohemian Rhapsody
```

### 🎧 Распознавание музыки (Shazam)
Отправьте голосовое сообщение или аудиофайл, и бот распознает трек.

### ❤️ Сохранение в плейлист
Нажмите ❤️ на любом треке, чтобы добавить его в свой плейлист.

### � Поделиться треком
Используйте кнопку "Поделиться" чтобы отправить трек друзьям.
